Class terms
Class The blueprint
Defines what a NUmart object looks like and what it can do. Everything lives inside it.
class NUmart: # whole thing = the class def __init__(self): ... def apply_promo(self, code): ...
Object / Instance Created from the class
The actual live store built from NUmart. Has its own cart, discount, and promo data.
# store = the object store = NUmart() store.discount # → 0 store.promo_used # → ""
Constructor Runs on create automatically
The __init__ method. Fires the moment you write NUmart(). Sets all starting values.
def __init__(self): self.discount = 0 self.promo_used = "" self._load_inventory()
self Refers to the current object
Always first in any method. Python fills it automatically — you NEVER type it when calling.
def apply_promo(self, code): self.discount = 15 store.apply_promo("SAVE10") # Python does: apply_promo(store, "SAVE10")
Attribute / Property Data stored inside the object — self.something
Your two key attributes. They survive between method calls — set in one method, readable in another.
self.discount = 0 # attribute — stores the % number self.promo_used = "" # attribute — stores the code string store.apply_promo("NUFV30") store.discount # → 30 (changed by apply_promo) store.remove_promo() store.discount # → 0 (reset by remove_promo)
Method Function that belongs to a class
Your 4 functions are all methods of NUmart. Always have self as first param.
def apply_promo(self, code): ... def remove_promo(self): ... def get_final_total(self): ... def display_price_summary(self): ...
Instantiation Creating an object from a class
The single line in main.py that creates the store. Calls the class like a function.
# This line is instantiation: store = NUmart() # ↑ triggers __init__ immediately # store is now a live NUmart object
Parameter Placeholder in the definition
Variable name written in the def line. Just a placeholder — no value yet.
def apply_promo(self, code): # ^^^^ # parameter (no value yet)
Argument Real value passed when calling
The actual value you give. Fills the parameter when you call the method.
store.apply_promo("STUDENT15") # ^^^^^^^^^^^ # argument (real value)
