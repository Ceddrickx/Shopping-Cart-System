## Group Member Task Distribution
# MP5: Shopping Cart System with Payment Handling
# Member 1 - Project Lead / Item Model & Main Menu
·       Define the Item class (ID, name, category, price, quantity, expiration)
·       Search item by name or ID (Search module)
·       Update item details (Update module)
·       Build the main menu and integrate all modules (main.py)
·       Data Structure: Linked List — item catalog/inventory
# Member 2 - Cart Operations & Undo Feature
·       Add item to cart (Insert module)
·       Remove item by name or ID (Delete module)
·       Display cart contents (Display module)
·       Undo last add/remove operation using stack transactions
·       Data Structure: Stack — undo transaction history
# Member 3 (Kim) - Price Calculation & Promo Codes
·       Compute total price of all items in cart
·       Apply promo code or discount
·       Display final payable amount (subtotal, discount, total)
·       Data Structure: Queue — promo code processing
# Member 4 (Ecks) - Payment Simulation 
·       Cash payment flow and change computation
·       Card payment flow (card number input, validation)
·       E-Wallet payment flow (account/reference input)
·       Display payment confirmation screen
·       Data Structure: Stack — payment method history
# Member 5 - Receipt, Transaction History & Error Handling
·       Generate formatted receipt summary on terminal
·       Store and display past transaction history
·       Handle empty structure operations and boundary conditions
·       Data Structure: Circular Linked List — transaction log / receipt history
